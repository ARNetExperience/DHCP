Router Configuration Commands

Configuring Pool for network 1
  • ip dhcp pool POOL_NAME1
  • network 192.168.1.0 255.255.255.0
  • default router 192.168.1.1
  • dns-server 8.8.8.8
  • exit
Configuring Pool for network 2
  • ip dhcp pool POOL_NAME2
  • network 192.168.2.0 255.255.255.0
  • default-router 192.168.2.1
  • exit

Explanation of Commands

  1. ip dhcp pool POOL_NAME: This command creates a pool of IP addresses that can be assigned automatically to devices on a network.
  2. network 192.168.1.0 255.255.255.0: This command tells the network which range of IP addresses it can give out from the "POOL_NAME" box.
  3. default router 192.168.1.1: This command tells the devices on the network which candy jar to go to when they want to go out of the network.
  4. dns-server 8.8.8.8: This command tells the devices on the network where to look when they need to find a website.
  5. exit: This command is like saying "I'm done" or "That's all" to the network.
  1. ip dhcp pool POOL_NAME2: Similar to the previous pool, this command creates another pool of IP addresses named "POOL_NAME2".
  2. network 192.168.2.0 255.255.255.0: Specifies the range of IP addresses for the "POOL_NAME2" box.
  3. default-router 192.168.2.1: Specifies the default router for devices using IP addresses from "POOL_NAME2".
  4. exit: Indicates the end of configuration for "POOL_NAME2".
